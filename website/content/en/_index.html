What is Kubeflow on vSphere?

Kubeflow on vSphere is an open source distribution of Kubeflow that allows customers to build machine learning systems with ready-made vSphere service integrations. Use Kubeflow on vSphere to streamline data science tasks and build highly reliable, secure, and scalable machine learning systems with reduced operational overheads.

Enterprise Ready
  • Multi-user and authentication support
  • Distributed training jobs and serving systems
  • Multi-framework: TensorFlow, PyTorch, Apache MXNet, and more
  • Pipeline for rapid and reliable ML workflows
Multi-Cloud Support
  • Easy installation and deployment on latest vSphere 8 with TKG
  • vSphere App platform support
  • Based on Tanzu Kubernetes Grid 2.0
  • Ready to run on any VMware Cloud
No Limitation
  • Free to use
  • Open-source project
  • Responsive Slack channel with experts
  • More features under development

Open Source

vSphere is adding value to the open source Kubeflow project by providing its own distribution: Kubeflow on vSphere. Follow along with or contribute to development in the Kubeflow on vSphere repository.
